Section 1 Overview
Originator Metropolitan Council - Metro Transit
Title Transitway Alignments
Abstract This dataset depicts the alignment of all existing transitway alignments. Transitways include light rail, commuter rail and Bus Rapid Transit (BRT) alignments. The geography was created by using heads-up digitizing referencing aerial photography.
Purpose To have a general depiction of the transitway alignments.

Section 2 Data Quality
Attribute Accuracy Station origin and destination segments are generally reflective the branded name of the station. Mileages are based on the GIS software measurements of the polylines.
Logical Consistency The dataset is topologically 'clean'.
Completeness Entire corridor is covered.
Horizontal Positional Accuracy This dataset was created using heads-up digitizing from aerial orthophotography.
Lineage The dataset was created by using heads-up digitizing referencing aerial photography.

Section 5 Attributes
Overview ATTRIBUTES:

route: A string identifying the number associated with the route and its schedules.

routesort: The route number of the transitway.

Direction: The general direction of the transitway alignment.

FromStat: The origin station of a station to station segment.

ToStat: The destination station of a station to station segment.

Miles: The estimated mileage of the segment

Name: Transitway name

Transitway: The transitway name (blue line, red line, etc.)

TrackID: A unique identification number for each track segment

Type: The transportation type used on the transitway
